# CounterDom

A simple DOM-based counter web application built with HTML, CSS, and JavaScript.  
Users can increment, decrement, and reset a counter value dynamically.

## Features

- Increase counter
- Decrease counter
- Reset counter
- Interactive and responsive UI

## How to Use

1. Clone the repository or download the project.
2. Open `index.html` in your browser.
3. Use the buttons to increment, decrement, or reset the counter.

## Technologies Used

- HTML
- CSS
- JavaScript
- DOM Manipulation
